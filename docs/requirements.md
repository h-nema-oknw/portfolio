# 根間大輝 ポートフォリオ — 要件定義書

## プロジェクト概要

| 項目 | 内容 |
|---|---|
| プロジェクト名 | 根間大輝 ポートフォリオサイト |
| クライアント | 根間大輝（個人） |
| 目的 | 転職活動・副業獲得における営業ツールとして機能するポートフォリオを GitHub Pages で公開する |
| Notion ページ | 未設定（後日「Private_プロジェクト管理 > 002_ポートフォリオ作成」に追加予定） |

---

## ゴール・成功基準

### 最重要ゴール（最優先）
転職活動・副業獲得の場面で先方に共有できる、デザイン性と個性を兼ね備えたポートフォリオを GitHub Pages（`~/portfolio`）で公開する

### マイルストーン別ゴール

| 時点 | 目標状態 |
|---|---|
| 公開時 | GitHub Pages で正常公開。Three.js パーティクル・GSAP アニメーションが動作し、デザイン性の高いページが完成している |
| 公開後 | 永続運用。経歴・スキル・実績を不定期に追加・改修できる構成になっている |

### KPI（測定指標）
現時点では数値目標は設定しない

### 失敗の定義（避けたい状態）
- GitHub Pages で公開できない状態
- アニメーション・インタラクションのない静的なだけのページ
- デザイン性・個性が低い凡庸な仕上がり

---

## 要件（実装する）

### 必須機能・コンテンツ
- ヒーローセクション（氏名・肩書き・グリッチアニメーション・Three.js パーティクル背景）
- About セクション（自己紹介・実績数値カード）
- スキルセクション（言語・ツール・AI活用の紹介）
- 経歴セクション（職歴・資格・資格）
- 制作実績セクション（Works）- サンプル成果物の掲載（他の GitHub Pages URL へのリンク・埋め込み）
- リンク一覧（SNS・関連企業など）
- メールアドレス掲載（お問い合わせフォームの代替）
- ダークモード切替
- PC・スマートフォン両対応（レスポンシブデザイン）
- プロフ写真・ロゴ・アイコン差し込みのプレースホルダー

### デザイン・UX
- アースカラー（ウォームゴールド・テラコッタ・オリーブ）× ブラックの配色
- HIPHOP × IT／デジタルを融合した洗練されたクールなトーン
- Three.js パーティクル背景（マウス追従・スクロール連動）
- GSAP ScrollTrigger によるスクロールアニメーション
- グリッチエフェクト（ヒーローテキスト）
- スキャンライン・ノイズオーバーレイ
- モノスペースフォント活用（ターミナル・コード風 UI）

---

## 非要件（実装しない）

- ブログ機能（今フェーズはスコープ外）
- お問い合わせフォーム（メールアドレス掲載で代替）
- 今後の営業・未来の計画に関するコンテンツ
- バックエンド・データベース・サーバーサイド処理
- CMS・管理画面

---

## ターゲット・ユーザー

| 項目 | 内容 |
|---|---|
| メインターゲット | 不特定多数（主に IT 企業採用担当・副業依頼者） |
| 利用場面・動機 | 求人応募時の URL 共有、面接中に先方に見せる、副業営業時の自己紹介 |
| 利用環境 | PC・スマートフォン両対応 |

---

## 制約・前提条件

- ホスティングは GitHub Pages に限定（静的ファイルのみ）
- リポジトリ URL: `~/portfolio`（`username.github.io/portfolio`）
- 開発・保守は根間本人（1名）
- 公開希望日は未定（早いほど良い）
- コーディング規約・命名規則の指定なし（標準慣例に準拠）
- プロフ写真・ロゴ・アイコンは後から差し込む（プレースホルダーで対応）
